Sources & notes

Opening ratings: model Elo-style estimates (June 11, 2026); groups, bracket and Annex C third-place allocation per the official FIFA format; opener Mexico 2–0 South Africa locked as played. “Refresh results” asks Claude (with web search) for final scores only; everything returned is validated against the fixture list before it touches the model. Live or in-progress games are ignored. Inside Claude the fetch is built in; on the public web the page calls the Anthropic API directly with a key you provide, which never leaves your browser. The third-place match is not modelled. Group tie-breaks on identical points/GD/GF are resolved by lot in-simulation — an approximation of FIFA's full criteria. Knockout results are applied in every simulated bracket where that pairing occurs. This is a model, not advice. A 19% favourite loses four times out of five.
